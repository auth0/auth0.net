Interface IManagementConnection

Interface for connectivity between ManagementApiClient and the remote server.

Namespace: Auth0.ManagementApi
Assembly: Auth0.ManagementApi.dll
Syntax
public interface IManagementConnection
Remarks

A default implementation of this using is included as HttpClientManagementConnection. You should mock this interface to perform unit testing of code that uses ManagementApiClient.
