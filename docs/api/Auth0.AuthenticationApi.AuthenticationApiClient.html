Parameters
Type Name Description
System.String domain

Your Auth0 domain name, e.g. tenant.auth0.com.
IAuthenticationConnection connection

Optional IAuthenticationConnection used to influence connection behavior. Defaults to a freshly created HttpClientAuthenticationConnection that uses a single System.Net.Http.HttpClient.
Remarks

To use a custom System.Net.Http.HttpClient or System.Net.Http.HttpMessageHandler create a HttpClientAuthenticationConnection passing that into the constructor. e.g.

var client = new AuthenticationApiClient(new HttpClientAuthenticationConnection(myHttpClient));
or 
var client = new AuthenticationApiClient(new HttpClientAuthenticationConnection(myHttpMessageHandler));

The grant_type parameter required by the /oauth/token endpoint will automatically be inferred from the request parameter. If no Realm was specified, then the grant_type will be set to "password".
